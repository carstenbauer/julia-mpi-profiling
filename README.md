# Julia MPI Profiling

In this repository we want to demonstrate and asses how different HPC tools can be used to "profile" MPI-parallel Julia applications.

## Potential tools
* [Extrae](https://tools.bsc.es/extrae)
* [Score-P](https://www.vi-hps.org/projects/score-p/)
* [Intel VTune Profiler](https://www.intel.com/content/www/us/en/developer/tools/oneapi/vtune-profiler.html#gs.4zeoog)
* [NVIDIA Nsight Systems](https://developer.nvidia.com/nsight-systems)
* [Tau](https://www.cs.uoregon.edu/research/tau/home.php)
* [HPCToolkit](http://hpctoolkit.org/)
